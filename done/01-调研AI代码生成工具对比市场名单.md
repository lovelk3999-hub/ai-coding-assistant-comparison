# 01. 调研AI代码生成工具对比市场名单

## 任务描述
列出15个主流 AI coding assistant 工具，作为本项目的首批数据源。

## 详细步骤

### Step 1: 搜索工具名单
用 chrome:control-chrome 或 agent-reach 搜索：
- "best AI coding assistant 2026"
- "top AI coding assistant tools"
- "AI coding assistant comparison"

### Step 2: 记录每个工具的信息
对每个找到的工具，记录：
| 字段 | 说明 |
|------|------|
| name | 工具名称 |
| url | 官网URL |
| category | 分类 |
| description | 一句话定位 |

### Step 3: 输出文件
写入 doing/01-tools-list.md，markdown表格格式

### Step 4: 参考
看 ai-video-tools-comparison/data/tools.json 了解分类方式

## 预期产出
- doing/01-tools-list.md（15个工具的完整表格）

- [ ] 每个工具都有官网URL
