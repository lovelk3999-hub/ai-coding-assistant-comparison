# 02. AI代码生成工具定价数据

> 来源：各工具官网定价页（Jina AI reader采集）+ Tembo对比文章
> 采集日期：2026-06-19

---

## 1. GitHub Copilot
- **官网**: https://github.com/features/copilot
- **Free**: 有限补全和聊天（2000补全/月，50聊天/月）
- **Pro**: $10/月 — 无限补全，多模型支持（GPT-4o, Claude, Gemini）
- **Business**: $19/用户/月 — 组织管理，IP豁免，隐私模式
- **Enterprise**: $39/用户/月 — SSO, 审计日志, 自定义策略

## 2. Cursor
- **官网**: https://www.cursor.com/pricing
- **Hobby (Free)**: 有限Agent请求 + 有限Tab补全
- **Pro**: $20/月 — 扩展Agent配额，前沿模型，MCP/Skills，Cloud agents
- **Pro+**: $40/月 — 更高配额
- **Ultra**: $60/月 — 最高配额
- **Teams Standard**: $40/用户/月 — 团队管理, Bugbot, SSO
- **Teams Premium**: $60/用户/月
- **Enterprise**: 联系销售 — 池化用量, SCIM, 审计日志

## 3. Windsurf (Codeium)
- **官网**: https://codeium.com/pricing
- **Free**: $0 — 有限Agent配额，有限模型，无限内联编辑，无限Tab补全
- **Pro**: $20/月 — 增加配额，前沿模型（OpenAI/Claude/Gemini），Cloud agents
- **Max**: $200/月 — 显著更高配额
- **Teams**: $80/月(团队计划) + $40/月/全Dev席位 — 无限成员，中央结算
- **Enterprise**: 联系销售 — SSO, 管理员控制, 专属部署

## 4. Claude Code (Anthropic)
- **官网**: https://claude.com/pricing
- 包含在Claude订阅中
- **Claude Pro**: $20/月 — 包含Claude Code访问
- **Claude Team**: $30/用户/月 — 团队版
- **Claude Enterprise**: 联系销售 — 企业级
- 也支持第三方提供商（BYO API Key）

## 5. OpenAI Codex CLI
- **官网**: https://openai.com/codex
- 包含在ChatGPT订阅中
- **ChatGPT Plus**: $20/月 — 包含Codex CLI
- **ChatGPT Pro**: $200/月 — 更高使用限制
- **Enterprise**: 联系销售

## 6. Gemini Code Assist (Google)
- **官网**: https://cloud.google.com/gemini-code-assist
- **Free (个人)**: 慷慨免费层 — 最高180,000次补全/月 + 每日聊天配额
- **Gemini Code Assist Enterprise**: $22.80/用户/月 (年付) — 企业功能

## 7. Amazon Q Developer (AWS)
- **官网**: https://aws.amazon.com/q/developer/pricing/
- **Free Tier**: 慷慨免费层 — 代码补全 + 安全扫描（对AWS用户免费）
- **Pro Tier**: $19/用户/月 — 高级功能，高级安全扫描
- **Enterprise**: 联系销售

## 8. Tabnine
- **官网**: https://www.tabnine.com/pricing
- **Code Assistant**: $39/用户/月 (年付) — 代码补全 + AI聊天, 私有部署(SaaS/VPC/本地)
- **Agentic Platform**: $59/用户/月 (年付) — 包含Code Assistant全部 + 自主Agent, MCP, 治理控制
- 注：使用自有LLM时无限用量；用Tabnine提供的LLM时按实际消耗付费

## 9. JetBrains AI Assistant
- **官网**: https://www.jetbrains.com/ai/
- **AI Pro**: $10/月 — 在JetBrains IDE中的AI辅助
- **AI for Business**: 联系销售 — 团队管理
- 另有 Junie (AI配对编程) 单独定价
- JetBrains All Products Pack 订阅用户可享折扣

## 10. Cline
- **官网**: https://github.com/cline/cline
- **Free**: 开源，免费（自带API Key）
- 支持多种LLM提供商（OpenAI, Anthropic, Google等）
- 支持自托管

## 11. Aider
- **官网**: https://aider.chat
- **Free**: 开源，免费（自带API Key）
- Git原生配对编程工具
- 支持多种LLM

## 12. Devin (Cognition)
- **官网**: https://devin.ai/pricing
- **Free**: $0 — 有限Agent配额，有限模型
- **Pro**: $20/月 — 增加配额，前沿模型，Cloud agents
- **Max**: $200/月 — 显著更高配额
- **Teams**: $80/月 + $40/月/全Dev席位 — 团队协作
- **Enterprise**: 联系销售

## 13. Sourcegraph Cody
- **官网**: https://sourcegraph.com/pricing
- **Free**: 有限使用（搜索公开代码库）
- **Enterprise**: 联系销售 — 深度代码搜索，批量变更，代码洞察
- 注：Sourcegraph目前主要面向企业，定价需联系销售

## 14. Factory (Factory Droid)
- **官网**: https://www.factory.ai/pricing
- **Hobby**: 免费 — 桌面/CLI/SDK，本地后台Agent
- **Pro**: 月付 — ~5x Hobby用量，Droid Computers
- **Max**: 月付 — ~10x Pro用量，新功能抢先体验
- **Teams**: 联系销售 — 多成员(至150席位)，SSO, SAML/SCIM, 零数据保留
- 注：具体价格未公开，联系销售获取报价

## 15. Continue (已并入Cursor)
- **官网**: https://continue.dev
- **状态**: 已被 Cursor 收购，开源代码仍可用
- 原为开源IDE代码助手，支持自定义模型
