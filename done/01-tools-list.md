## 分类说明

| 分类 | 说明 | 工具数 |
|------|------|--------|
| IDE助手 | 安装在现有IDE中的AI插件 | 7 (Copilot, Gemini, Q, Tabnine, JetBrains, Cody, Continue) |
| AI原生IDE | 完全以AI为中心的代码编辑器 | 2 (Cursor, Windsurf) |
| CLI代理 | 终端中运行的AI编码代理 | 3 (Claude Code, Codex CLI, Aider) |
| 开源代理 | 开源/自部署的AI代理 | 2 (Cline, Continue也可归此类) |
| 自主代理 | 后台自主运行的AI程序员 | 1 (Devin) |
| 多面代理平台 | 面向多种工程任务的代理平台 | 1 (Factory) |
