# 02. 采集AI代码生成工具对比各工具官网定价

## 任务描述
对15个工具，逐一访问官网定价页，采集定价方案。

## 详细步骤

### Step 1: 确定采集方案
- 定价公开的官网 -> 用 chrome:control-chrome 直接访问
- 需登录才能看定价的 -> 记录'需登录'

### Step 2: 采集每个工具的定价
对每个工具，采集：
- 免费版（Free）的功能限制
- Pro/Standard版的价格和功能
- Team/Business版的价格
- Enterprise价格（按需报价的标'联系销售'）

### Step 3: Checkpoint
每采完5个工具，更新 doing/CHECKPOINT.md

### Step 4: 输出格式参考
参考 ai-video-tools-comparison/data/tools.json 中的 plans 数组格式
```json
"plans": [
  {"name": "Free", "price": "$0", "features": ["1 seat", "2000 completions"]},
  {"name": "Pro", "price": "$10/mo", "features": ["Unlimited"]}
]
```

## 预期产出
- doing/02-pricing-data.md（所有工具的定价数据）
