# 04. 采集AI代码生成工具对比第三方评分数据

## 任务描述
采集各工具的G2评分、Reddit讨论等第三方数据。

## 详细步骤

### Step 1: G2评分
用 chrome:control-chrome 搜索 'g2 [工具名]'
或使用 TinyFish API。采集 rating(如4.5/5)、reviews数量。

### Step 2: Reddit讨论
搜索 'site:reddit.com [工具名] review'
采集每条：title / url / score(点赞数) / comments数

### Step 3: 无数据处理
- 无G2页面的 -> 标'暂无G2数据'
- Reddit讨论少的 -> 搜'alternative'或'vs'

### Step 4: 参考
参考 ai-video-tools-comparison/scripts/collect_social_proof.py

## 预期产出
- doing/04-social-proof.md（评分和讨论数据）
